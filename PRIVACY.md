# Privacy Policy

## Data Collection and Usage

This application is designed for internal use only and accesses QuickBooks data for financial analysis purposes.

**What Data We Access:**
- Profit & Loss statements from your QuickBooks account
- Financial data necessary for generating reports

**How We Use Your Data:**
- Data is retrieved from QuickBooks API for analysis purposes only
- All data is stored locally in a private location
- Data is used solely for generating financial reports and analysis

**Data Storage:**
- All data is stored securely on your local system or private server
- No data is transmitted to third-party services (except QuickBooks API)
- Data is not shared with any external parties

**Data Security:**
- OAuth tokens are stored securely
- All API communications use HTTPS encryption
- Access is restricted to authorized users only

**Third-Party Services:**
- This application uses QuickBooks API (Intuit)
- We do not share your data with any other third parties
- QuickBooks API usage is subject to Intuit's privacy policy

**Your Rights:**
- You can revoke API access at any time through QuickBooks settings
- You can delete stored data at any time
- You maintain full control over your QuickBooks data

**Contact:**
For questions about this privacy policy, please contact the application administrator.
